# React Starter Project
---
Building my own react starter from scratch with Webpack, ES6, Babel, react-router and sass-loader.

Setup
---
`npm install`

Usage
---
1. `node server.js` or `npm start`
2. Open [http://localhost:3000/](http://localhost:3000/).

Compile
---
`npm run compile`
